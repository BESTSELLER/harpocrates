# Contributing to Harpocrates

First off, thank you for considering contributing to Harpocrates! It's people like you that make Harpocrates such a great tool.

## Where do I go from here?

If you've noticed a bug or have a feature request, make sure to check our [Issues](https://github.com/BESTSELLER/harpocrates/issues) if it has already been reported. If not, open a new issue using the provided templates!

## Creating a Pull Request

1. If you've added code, add tests!
2. Ensure your code lints by running `golangci-lint run`.
3. Document any changes in `README.md`.
4. Submit your pull request, and verify that the GitHub Actions CI passes.

Thanks for contributing!
